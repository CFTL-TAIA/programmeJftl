# US005_Navigation_Contextualisee_Test

## Cas de tests d'acceptance

1. Depuis l'accueil, un clic sur une conference ouvre `conferences/?id=...`.
2. Depuis l'accueil, un clic sur un speaker ouvre `speakers/?id=...`.
3. Depuis l'accueil, un clic sur une salle ouvre `salles/?id=...`.
4. Depuis l'accueil ou une page detaillee, un clic sur une entreprise ouvre `entreprises/?id=...`.
5. Sur la page d'un speaker, la liste de ses conferences contient des liens vers les conferences, salles et entreprises associees.
6. Sur la page d'une salle, la liste des conferences affiche l'horaire en premier puis des liens vers les speakers et entreprises associes lorsque presents.
