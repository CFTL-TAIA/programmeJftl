# US004 - Accueil Programme

## User Story

En tant que visiteur du projet API TAIA, je veux afficher sur la page d'accueil le programme complet de la JFTL 2026 afin de consulter rapidement tous les créneaux de la journée sans passer par l'espace API.

## Valeur attendue

- proposer une vraie experience front oriente evenement
- exposer le contenu reel de la BDD JSON au lieu d'exemples fictifs
- faciliter la navigation vers les pages detaillees du site
- distinguer visuellement les demos editeurs du reste du programme

## Criteres d'acceptation

1. La page d'accueil affiche le titre `PROGRAMME DE LA JFTL 2026`.
2. Le programme est charge depuis l'API reelle et couvre l'ensemble des créneaux de la journée.
3. Les compteurs conferences, speakers, salles et entreprises apparaissent sous le programme.
4. Le header expose les liens vers Swagger et vers le projet GitHub.
5. La page conserve un acces explicite vers l'espace API dedie.
6. Les demos editeurs sont affichees dans une colonne a droite, plus compacte que les autres sessions.
